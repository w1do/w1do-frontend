# SUMMARY

## Обзор проекта
**W1DO DIGITAL** — это профессиональный сайт AI-инженера и LLMOps специалиста. Сайт ориентирован на демонстрацию экспертизы в области внедрения искусственного интеллекта (Claude, Gemini, Junie) в бизнес-процессы.

## Основные цели
- Привлечение органического трафика по SEO-запросам, связанным с AI-автоматизацией.
- Демонстрация кейсов и услуг.
- Предоставление удобного интерфейса для связи и заказа услуг.

## Технологический стек
- **Astro**: Фреймворк для создания быстрых контент-ориентированных сайтов.
- **Tailwind CSS**: Утилитарный CSS-фреймворк для стилизации.
- **Directus**: Headless CMS для управления контентом (в процессе интеграции).
- **n8n**: Автоматизация генерации SEO-контента.
