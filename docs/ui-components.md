# Документация UI компонентов

В проекте используется компонентный подход на базе Astro и Tailwind CSS.

## Структура компонентов

Компоненты расположены в `src/components/`.

### Shared компоненты (`src/components/shared/`)
Общие компоненты, используемые на разных страницах:
- `header/`: Компоненты навигации и заголовка сайта.
- `footer/`: Подвал сайта с контактной информацией.

### Секции страниц (`src/components/`)
- `Hero.astro`: Главный блок первой страницы с основным оффером.
- `About.astro`: Блок "О компании/инженере".
- `Services.astro`: Список предоставляемых услуг по внедрению AI.
- `Process.astro`: Описание этапов работы (AI Roadmap).
- `Works.astro`: Портфолио выполненных проектов.
- `Testimonials.astro`: Отзывы клиентов.
- `Faqs.astro`: Ответы на часто задаваемые вопросы.
- `Awards.astro`: Блок с достижениями и наградами.
- `Choose.astro`: Преимущества работы с нами.
- `Cta.astro`: Призыв к действию.

## Использование компонентов

Пример подключения компонента в Astro:

```astro
---
import Hero from '../components/Hero.astro';
---

<Layout title="AI Инженер">
  <Hero />
</Layout>
```

## Стилизация
Используется **Tailwind CSS**. Все стили прописаны непосредственно в классах компонентов для обеспечения модульности и быстроты разработки.
