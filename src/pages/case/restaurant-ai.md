Инновационное решение для ресторанных сетей: прогнозирование спроса, оптимизация закупок и персонализированные рекомендации для гостей на базе ИИ.

### Описание решения
В рамках данного проекта была разработана и внедрена интеллектуальная система управления ресторанным бизнесом. Основной упор был сделан на минимизацию списаний продуктов и повышение лояльности посетителей через персонализацию предложений.

### Ключевые функции
- Прогнозирование спроса на основе машинного обучения
- Автоматический расчет заказов поставщикам
- Система персональных рекомендаций в мобильном приложении
- Интеграция с популярными POS-системами (R-Keeper, iiko)

### Задачи проекта
Основной задачей было свести к минимуму списания скоропортящихся продуктов при сохранении полного меню.

### Реализация
Проект включал в себя создание ETL-пайплайнов для сбора данных из всех точек сети в единое хранилище.
