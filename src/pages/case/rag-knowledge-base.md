Создание интеллектуального ассистента с использованием RAG (Retrieval-Augmented Generation) и OpenAI для быстрого поиска по документам.

### Описание решения
Мы разработали систему, которая позволяет сотрудникам мгновенно находить ответы в тысячах корпоративных документов (KDocs, PDF, Wiki). ИИ не просто ищет ключевые слова, а понимает смысл вопроса и дает конкретный ответ со ссылками на источники.

### Ключевые функции
- Поиск по всем внутренним источникам (Confluence, Google Drive, Notion)
- Сохранение контекста диалога
- Автоматическое обновление индекса при изменении документов
- Права доступа (ИИ отвечает только на основе разрешенных документов)

### Задачи проекта
Главной проблемой было обеспечение точности ответов и предотвращение галлюцинаций ИИ.

### Реализация
Мы использовали современный стек технологий для обеспечения максимальной скорости поиска.
