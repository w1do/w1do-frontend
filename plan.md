# План разработки контента для сайта AI-инженера

Этот план описывает стратегию наполнения сайта экспертным контентом на русском языке с учетом SEO и фокуса на бизнес-результат.

## 1. Анализ ключевых слов
Наиболее релевантные ключевые слова для интеграции:
- **Интеграция AI в бизнес процессы** (Основной интент)
- **Внедрение ИИ в бизнес** (Главный заголовок)
- **AI Roadmap** (Для блока "Процесс")
- **AI автоматизация бизнеса** (Для блока "Преимущества")
- **AI ассистенты / чат-боты** (Для блока "Услуги")
- **Конструктор ИИ ассистентов** (Для блока "Услуги")
- **Как внедрить ИИ в свой бизнес** (Для блока "FAQ" или "Блог")
- **Результат и окупаемость** (Ключевой посыл)

## 2. Структура наполнения (content.md)
Каждый блок будет проработан в файле `content.md` по следующей схеме:
1. **Цель блока**: Какой интент закрывает.
2. **Заголовки и тексты**: Экспертный контент с вкраплением ключевых слов.
3. **Техническое задание по изображениям**: Обязательные `alt` теги для каждого `<img>`.
4. **Призыв к действию (CTA)**: Конверсионные элементы.

## 3. Этапы реализации
1.  **Сборка текстов**: Написание экспертных текстов для всех 9 услуг.
2.  **SEO-оптимизация**: Проверка плотности ключевых слов без переспама.
3.  **Аудит медиа**: Описание всех необходимых изображений и их описаний (alt).
4.  **Финальная сборка**: Подготовка финального `content.md` для внедрения.
